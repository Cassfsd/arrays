Indice: comienza en cero. Lenght:la cantidad de elementos del array. Métodos: -Metodo pop():elimina el ultimo elemento del array.

-Método push ():agrega el elemento al final del array.

-Método shift()remueve el primer elemento del array.

-Método unshift():agrega un elemento en la primera posición del array.

-Método indexOf()retorna la posición del elemento buscado.

-Método includes():devuelve true si el valor buscado se encuentra en el array.

-Método to string:cinvierte el array en una única cadena separada por una coma. -Método join():convierte el array en una cadena separada por lo que le indiques.

-Método concat():une los arrays seleccionados en un único array.
